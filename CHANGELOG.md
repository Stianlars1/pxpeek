## [0.1.0] — 2026-04-15

### Added
- First release.
- Inline pixel-equivalent inlay hints for 18 relative CSS units.
- Support for CSS, SCSS, SASS, and LESS file types.
- Per-language toggle via *Settings → Editor → Inlay Hints → Values → Pixel equivalents*.
